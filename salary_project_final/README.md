# Employee Salary Prediction Project

## Project Overview
This project uses a Random Forest Regressor to predict employee salaries based on demographic and professional features.

## Dataset
- **Source:** Synthetically generated (2,500 records).
- **Features:** Age, Education, Experience, Skills Score, Certifications, Job Role, Company Type.

## Performance
- **R" Score:** 0.8864
- **MAE:** ₹9,561.86

## How to Run
1. Install dependencies: `pip install pandas numpy scikit-learn matplotlib seaborn`
2. Run the script: `python main.py`
